Quick Guide Doxygen

References

https://www.doxygen.nl/manual/starting.html

Install and run Doxygen

sudo apt install doxygen
sudo apt install doxygen-gui

mkdir doxygen
cd doxygen
# create Doxyfile and edit manually
doxygen -g
# or run qwizzard and finally save the Doxyfile 
doxywizard
# if the configuration is done generate documentation
doxygen
